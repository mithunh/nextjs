# Uzofin

**Uzofin** is a modern, high-performance web application built with **Next.js**, **React**, and **Tailwind CSS**. This project leverages the latest frontend technologies to deliver a responsive, fast, and visually appealing user experience.

## Project Overview

Uzofin is designed to provide a scalable frontend architecture using **Next.js 16** with **React 19**, optimized for performance, responsiveness, and modern UI/UX patterns.  

Key goals of this project:  
- Fast, server-side rendered pages with Next.js.  
- Reusable React components for maintainability.  
- Tailwind CSS for utility-first, responsive styling.  
- Integration with modern animation and UI libraries like **Framer Motion** and **Swiper**.  

---

## Features

- Fully responsive design using Tailwind CSS.  
- Smooth animations and transitions with Framer Motion.  
- Interactive sliders and marquees using Swiper and React Fast Marquee.  
- TypeScript support for type safety and better developer experience.  
- ESLint configured to maintain code quality and consistency.

## Tech Stack

**Frontend:**  
- [Next.js](https://nextjs.org/) v16.0.5  
- [React](https://reactjs.org/) v19.2.0  
- [TypeScript](https://www.typescriptlang.org/) v5  

**Styling & UI:**  
- [Tailwind CSS](https://tailwindcss.com/) v4  
- [Framer Motion](https://www.framer.com/motion/) v12.23.25  
- [Swiper](https://swiperjs.com/) v12.0.3  
- [React Fast Marquee](https://www.npmjs.com/package/react-fast-marquee) v1.6.5  
- [React Icons](https://react-icons.github.io/react-icons/) v5.5.0  

**Development Tools:**  
- ESLint for linting and code quality  
- Babel plugin for React compilation  
- PostCSS for Tailwind processing

## Installation

```bash

cd uzofin
npm install
npm run dev

| Script          | Description                       |
| --------------- | --------------------------------- |
| `npm run dev`   | Runs the app in development mode  |

| `npm run start` | Starts the production server      |
| `npm run lint`  | Runs ESLint to check code quality |
